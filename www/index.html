Cassius

Automated Reasoning for Web Page Layout

Cassius is a new tool to automatically reasoning about web page layout. It includes a formal specification for a fragment of CSS 3.0 and a translation of that specification to SMT (using quantifier-free linear real arithmetic).

Cassius can be used to verify, debug, and synthesize stylesheets, all without user intervention.

Status

Cassius supports the properties that define the following CSS concepts:

  • width
  • height
  • padding
  • border
  • margin
  • position
  • float
  • text-align
  • box-sizing
