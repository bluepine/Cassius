Cassius

Automated Reasoning for Web Page Layout

Reasoning about CSS and browser layout is hard for web developers, browser engineers, and tool developers. Cassius is a project from the University of Washington to automatically reason about web page layout. Cassius can verify stylesheets against specifications, reduce layout bugs to the relevant lines in a stylesheet, or generate a stylesheet to achieve a certain layout, all without user intervention.

Status

Cassius understands a subset of the full CSS 3 standard, mostly containing the basic box model and floating boxes, and passes some sections of the CSS WG conformance tests. Cassius has been submitted to PLDI'16.
