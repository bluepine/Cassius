    • UW CSE’s Programming Languages and Software Engineering group advances fun­damental research and prac­tical appli­cations in program­ming environ­ments, program anal­ysis, language design, run­-time sys­tems, compilers, testing, verifi­cation, and security. We have strong ties to research­ers from Cray, Microsoft Research, NVIDIA, and others. Several of us also participate in the SAMPA group.

    Projects

    • Automatic Memory Models

      Automatically synthesizing specifications for relaxed memory consistency.

    • Approximate Synthesis

      Discovering the best balance of quality and efficiency, automatically.

    • Bagpipe

      Bagpipe verifies the correctness of BGP router configurations.

    • Cassius

      Automatically generate CSS to make a webpage match a mock-up.

    • Crust

      Verify memory invariants for Rust unsafe blocks.

    • Data Structure Synthesis

      Synthesizing high-performance data-structure implementations.

    • dbcert

      A library for writing and verifying crash-resistant applications.

    • defects4j

      Testing the tests that test your tests with a database of real bugs from open-source projects.

    • Herbie

      Find more accurate ways to write a floating point expression, automatically.

    • Incarnate

      Reason about objects in the real world by reasoning about the programs that print them.

    • Neutrons

      Verifying the control software for a radiation device used to treat cancers of the throat.

    • Retroactive Debugging

      Capture how a program executes, then go back in time with the debugger to see what happened.

    • Staccato

      Dynamically finding bugs in software configuration update code.

    • Timelapse

      The first user interface for capturing and replaying web application executions directly from the browser's developer tools.

    • Verdi

      Prove distributed systems correct in Coq, and extract them into fast running code.

    • Verso

      Prove assembly-level peephole optimizations correct in CompCert.

    • Timing Models

      Automatically detecting side-channel attacks.
